Funding diagram — more directions

Same story: ₹2.5 Cr Goal, ₹1 Cr Raised (40%), ₹20 L Utilised (8% of goal). Four fresh, thematic ideas.

Option I

Diamond fill — Jubilee tie-in

A diamond (nod to the Class of 1965 Diamond Jubilee) that fills to the raised level, with the utilised band at its base.

Option J

Summit climb

An aspirational ascent — base is what is utilised, the flag marks how far we have climbed (40%), the peak is the goal.

Option K

Coin stack

A literal stack — each coin ≈ ₹25 L. Four coins raised so far; the base coin marks what is already utilised; faint coins are still needed.

Option L

Milestone track

A horizontal journey with flags — utilised, raised (you are here), and the goal at the finish.
